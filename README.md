# NASA Meteorite Analysis

Analyzing NASA's Meteorite Landings dataset through data cleaning, feature engineering, geospatial analysis, and visualization.

## Overview

This project explores meteorite mass, classifications, recovery patterns, geographic distribution, and historical trends using Python and data science tools.

## Dataset

The dataset contains information about meteorite landings collected and maintained by NASA. It includes meteorite characteristics, landing locations, recovery details, and temporal information.

## Features

- Data cleaning and preprocessing
- Feature engineering
- Geospatial analysis
- Visualization and statistical analysis
- Historical trend analysis

## Files

- `Nasa_Meteorite.ipynb`: Jupyter notebook containing the analysis
- `meteorite_ds.csv`: NASA meteorite dataset
- `README.md`: This file
- `LICENSE`: Project license

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Getting Started

1. Clone or download the repository
2. Install required dependencies
3. Open `Nasa_Meteorite.ipynb` in Jupyter Notebook
4. Run the cells to reproduce the analysis
